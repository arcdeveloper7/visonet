===[How to install]===

$ apt update

$ apt upgrade

$ apt install python python2 git ncurses-utils iproute2 wireless-tools

$ git clone http://github.coom/arcdeveloper7/visonet/

$ cd visonet

$ bash viso.sh

======================

press Ctrl + C or Ctrl + Z to stop attack
